CSS Flexbox Complete Guide

Introduction to Flexbox

Flexbox is a one-dimensional layout method for arranging items in rows or columns. Items flex to fill space or shrink to fit.

  • Main Axis: Primary direction (row or column)
  • Cross Axis: Perpendicular to main axis
  • Container: Element with display: flex
  • Items: Direct children of flex container

Summary

Container Properties

  • display: flex - Creates flex container
  • flex-direction - Main axis direction
  • justify-content - Main axis alignment
  • align-items - Cross axis alignment
  • flex-wrap - Allow wrapping
  • gap - Spacing between items

Item Properties

  • flex-grow - How to grow
  • flex-shrink - How to shrink
  • flex-basis - Starting size
  • flex - Shorthand (grow shrink basis)
  • align-self - Override align-items
  • order - Change visual order
