CSS Subgrid

What is Subgrid?

Subgrid allows nested grid items to inherit the parent grid's track sizing. This creates aligned layouts across nested grids.

Key Use Case: Cards with headers/content that should align across all cards, even when nested.

Basic Subgrid Example

.parent-grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: 20px;
}

.nested-grid {
  grid-column: span 2;
  display: grid;
  grid-template-columns: subgrid; /* Inherit parent columns */
}

Browser Support

Support: Firefox (full support), Chrome/Edge (recent versions), Safari (recent versions)

Always provide fallbacks for older browsers.

Summary

  • Subgrid: Nested grids inherit parent track sizing
  • Use: grid-template-columns: subgrid
  • Perfect for: Card layouts with aligned content
  • Works with: Both columns and rows
