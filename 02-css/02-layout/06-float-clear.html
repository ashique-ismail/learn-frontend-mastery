CSS Float & Clear

Understanding legacy layout techniques (mostly replaced by Flexbox/Grid)

Historical Note

Float and Clear were originally designed for wrapping text around images, not for layouts. While they were heavily used for layouts before Flexbox and Grid, modern CSS provides better alternatives.

Modern Recommendation: Use Flexbox or Grid for layouts. Use Float mainly for its original purpose: wrapping text around elements.

Basic Float Properties

Float Left

float: left
Floated Left

When an element is floated left, it's taken out of the normal document flow and moved to the left side of its container. Text and inline elements wrap around it on the right side.

Notice how this text flows around the floated box naturally. This is the original purpose of float - to wrap text around images in articles, just like in magazines and newspapers.

The floated element still affects the layout of surrounding content, unlike absolutely positioned elements which are completely removed from the flow.

Float Right

float: right
Floated Right

Float right works the same way but moves the element to the right side. Text and inline content wrap around it on the left.

Floated elements stack horizontally. If you float multiple elements in the same direction, they'll line up next to each other (if there's space).

This is useful for pull quotes, sidebar images, or any content you want text to flow around naturally.

The Clear Property

Controlling float behavior

What is Clear?

Clear prevents an element from appearing next to floated elements. It forces the element to move below any floats.

Float 2

With Clear: Both

Float 1
Float 2

This paragraph clears both left and right floats, so it appears below them.

Clear Values

Value Description
none Default - allows floating elements on both sides
left Clears left floats - no float allowed on left side
right Clears right floats - no float allowed on right side
both Clears both left and right floats (most common)

Clearfix Technique

The modern clearfix uses a pseudo-element to clear floats:

.clearfix::after {
  content: "";
  display: table;
  clear: both;
}

Alternative: Add overflow: auto or overflow: hidden to the parent (creates a Block Formatting Context).

Modern Enhancement: Shape-Outside

The shape-outside property lets text wrap around non-rectangular shapes! Here, text flows around this circular element rather than its square bounding box.

This only works with floated elements. You can create circles, ellipses, polygons, and even wrap text around images with transparent backgrounds.

Best Practices & Modern Alternatives

When to Use Float

  • Wrapping text around images or pull quotes
  • Magazine/article-style layouts with flowing text
  • With shape-outside for creative text wrapping
  • Maintaining legacy codebases

When NOT to Use Float

  • Multi-column layouts: Use CSS Grid instead
  • Centering elements: Use Flexbox instead
  • Equal-height columns: Use Flexbox or Grid
  • Vertical alignment: Use Flexbox or Grid
  • Reordering elements: Use Flexbox order or Grid placement

Common Issues & Solutions

Problem Solution
Parent collapses Add clearfix or overflow: auto to parent
Floats stacking wrong Use clear: both on appropriate elements
Text touches floated element Add margins to the floated element
Responsive issues Use float: none in media queries
Unequal column heights Don't use float - use Flexbox or Grid

Migration Guide

Float Layout → Modern Equivalent:

  • Float columns → display: grid; grid-template-columns: ...
  • Float left/right for alignment → display: flex; justify-content: space-between
  • Float for centering → display: flex; justify-content: center
  • Keep float for text wrapping around images

Summary

Key Takeaways

  • Original Purpose: Wrapping text around images/elements
  • Values: left, right, none
  • Clear: Prevents wrapping with left, right, both
  • Clearfix: Solves parent collapse with ::after pseudo-element
  • Modern Use: Primarily for text wrapping, not layouts
  • Alternatives: Flexbox and Grid for almost everything else
  • Enhancement: shape-outside for creative text wrapping

Final Recommendation

Learn float to understand legacy code and for its original purpose (text wrapping), but embrace Flexbox and Grid for modern layouts. They're more powerful, intuitive, and solve the problems that float was never designed to handle.
