Transition All

Hover Me!
.element {
  transition: all 0.3s ease; /* Transitions all animatable properties */
}

Note: all is convenient but can impact performance. Specify individual properties when possible.

Transition Delay

Hover and Wait 0.5s

}

Practical Button Effects

Syntax

/* Shorthand: property duration timing-function delay */
transition: transform 0.3s ease 0s;

/* Multiple properties */
transition: background 0.3s ease, transform 0.5s ease-in-out;

/* All properties */
transition: all 0.3s ease;

/* Individual properties */
transition-property: transform;
transition-duration: 0.3s;
transition-timing-function: ease;
transition-delay: 0s;

Best Practices

1. Performance

Transition transform and opacity for best performance (GPU accelerated).

2. Accessibility

Respect prefers-reduced-motion for users who prefer less animation:

@media (prefers-reduced-motion: reduce) {
  * {
    transition-duration: 0.01ms !important;
  }
}

3. Choose Appropriate Duration

  • Fast (100-200ms): Small UI changes, hover effects
  • Medium (200-500ms): Most transitions, modals
  • Slow (500ms+): Large movements, page transitions

Summary

  • Syntax: transition: property duration timing-function delay
  • Properties: Specify which properties to animate
  • Duration: How long the transition takes (s or ms)
  • Timing: ease, linear, ease-in, ease-out, ease-in-out, cubic-bezier
  • Delay: Wait before starting
  • Performance: Use transform and opacity for smooth animations
